Fireback workspaces is the almost 95% of Fireback project, which is responsible for administration
functionality, codegen functionality, and provides structs and functions which workspaces itself, and other
generated code depend on it.

All of them are a part of workspaces, allthough it might be counter intuitive to common belief 
of separating responsibilities, this is chosen for simplicity.

In this section, we are going go deeper in code and functionality.
